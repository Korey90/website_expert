## Risks and Dependencies
- **Site speed / technical debt** — if month-1 audit reveals major issues, first 4–6 weeks focus on fixes, not content
- **Content bottleneck** — without regular new content, keyword growth stalls after initial on-page gains
- **CMS edit access** — must be confirmed before onboarding; restricted CMS = limited on-page scope
- **Algorithm updates** — Google core updates can temporarily affect rankings; communicate this risk upfront
- **Link building timeline** — organic link acquisition takes 3–6 months to show domain authority improvement

---

## Assumptions
- GA4 and GSC shared and verified before month-1 audit begins
- Client provides CMS login with editor-level access (or WE manages via developer access)
- Results are measured against agreed KPIs, not absolute ranking positions
- Reporting uses GSC + GA4 data only (no vanity metrics)
