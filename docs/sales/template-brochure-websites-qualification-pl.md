## Kryteria kwalifikacji

### Wymagania obligatoryjne (brak = deal-breaker)
- [ ] Budżet ≥ £799 (lub realistyczne oczekiwanie tego zakresu)
- [ ] Brak wymogu e-commerce ani uwierzytelniania użytkowników
- [ ] Zidentyfikowany decydent lub jasna ścieżka akceptacji
- [ ] Realistyczny harmonogram (minimum 3 tygodnie od podpisania zakresu)

### Czynniki podnoszące jakość dealu
- [ ] Gotowe zasoby brandingowe (logo, kolory, czcionki)
- [ ] Przygotowane lub sfinansowane treści (teksty + zdjęcia)
- [ ] Zainteresowanie planem opieki (`maintenance`, £149/mc)
- [ ] Zainteresowanie retainerem SEO (`seo`, £499/mc)
- [ ] Wcześniejszy nieudany projekt strony (tworzy pilność)

### Czerwone flagi
- 🚨 „Chcę to za 3 dni"
- 🚨 „Zapłacę, gdy strona zacznie zarabiać"
- 🚨 „Potrzebuję pełnego sklepu za £800"
- 🚨 „Potrzebuję 50 podstron i animacji"
- 🚨 Klient nie wie, kim są jego klienci
- 🚨 Wielu interesariuszy bez uzgodnionego briefu

---

## Zakres i granice
**Minimalny zakres (£799):** 5 podstron, projekt graficzny, CMS, formularz kontaktowy, podstawowe SEO, wsparcie 30 dni.

**Zakres wymagający ponownej kwalifikacji:**
- 10+ podstron → ponowna ocena harmonogramu i ceny
- Potrzeba copywritingu → dodaj usługę `content` lub zwiększ budżet projektu
- Sekcja blog / aktualności → uwzględnij szkolenie z CMS lub onboarding autora

## Ryzyka i zależności
- Brak potwierdzenia budżetu → ryzyko odrzucenia wyceny; zakotwicz w wartości przed podaniem ceny
- Decyzja przez komitet → ryzyko nieskończonych opóźnień bez project championa
- Brak treści → harmonogram się przesunie; należy uwzględnić fazę produkcji treści

---

## Założenia
- Lead pozyskany przez formularz na stronie, polecenie lub cold outreach
- Wstępna kwalifikacja odbywa się podczas 15–20 minutowej rozmowy lub wymiany e-maili
- Jeśli brakuje treści, traktuje się to jako osobny element zakresu lub przekierowanie do usługi `content`
